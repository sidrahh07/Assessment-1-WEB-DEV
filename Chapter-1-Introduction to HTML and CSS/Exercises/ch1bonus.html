html and css quick reference guide

html tags

<h1> - <h6> (Headings)

used for headings, h1 is the biggest and h6 is the smallest

<p>

used for a paragraph

<img >

used to add images

<a >

used to add links

<title>

gives a title to our webpage

<center>

puts the text in the center

<sup>

used to add superscript

<sub>

ussed to add subscript

<ul,>,<ol>,<li>

ul- for unordered lists
ol- for ordered lists
li - to add an item to the list

<marquee>

makes the text move in specified direction
default is from right to left

<strong>

makes the impotant text bolder

<acronym>

used for acronyms - when cursor hovers over the acronym, it shows the full form

CSS properties

color

sets the color

font-size

sets the size of the font

padding

to control the inner spacing of the element

margin

to control the outer spacing of the element

background-color

sets the background color of the element.

text-align

to adjust the alignment of text

border

adds a border of specified color to the element

text-decoration

to decorate the text
